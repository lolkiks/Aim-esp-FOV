# Aim-esp-FOV

Luau LocalScript для Roblox с базовыми функциями:
- ESP через `Highlight`
- FOV-круг через `Drawing`
- Aim Assist на ближайшую цель в радиусе FOV

## Файл
- `aim_esp_fov.lua`

## Управление
- `RightShift` — включить/выключить весь скрипт
- `E` — включить/выключить ESP
- `Q` — включить/выключить Aim
- `ПКМ (MouseButton2)` — удерживать для наведения (если включено в конфиге)

## Важно
Используйте только в собственных проектах/для тестирования механик.
